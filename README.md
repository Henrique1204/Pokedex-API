## Tutorial de criação de API com *NestJS*, *Fastify*, *Drizzle* e *PostegreSQL*.

### Criação do Projeto.

Execute o comando a seguir no terminal dentro da pasta do seu projeto.

```shell
npx @nestjs/cli new .
```

### Configuração do Fastify.

```shell
yarn add @nestjs/platform-fastify fastify
```
